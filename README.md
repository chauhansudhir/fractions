# 🍕 Fraction Fun!

An interactive, colorful, and kid-friendly React application designed to make learning fractions intuitive and fun. This app uses visual models like pizza slices, building blocks, and mixing bowls to help children understand how parts make a whole.

## 🚀 Features

The app includes **8 distinct interactive modes** to build a deep understanding of fractions:

1.  **🎈 Free Play:** Drag and drop colorful blocks into rows to compare their lengths visually.
2.  **🍕 Pizza Mode:** Build a pizza! Drag fraction slices onto a tray to see how they form a complete circle.
3.  **🚗 Journey Road:** Help a vehicle reach the finish line by adding fraction-based distances along the road.
4.  **🧱 Fraction Wall:** A visual breakdown showing how one "Whole" is divided into halves, thirds, fourths, sixths, and eighths.
5.  **⚔️ Battle (Comparison):** Side-by-side fraction "battles" where kids identify which fraction is larger, smaller, or equal.
6.  **📍 Number Line:** A draggable pin challenge to help kids translate "parts of a whole" into positions on a line.
7.  **🧩 Match (Equivalents):** A puzzle to find different combinations of small blocks that equal exactly 1/2.
8.  **🥣 Bake (Kitchen):** Follow a recipe! Add 1/4 and 1/2 cup scoops of flour to a bowl to reach a target measurement.

## 🎨 Design

- **Kid-Friendly UI:** Uses the rounded and playful **Nunito** font.
- **Glassmorphism:** Modern, soft-focus card designs with vibrant gradients.
- **Interactive Feedback:** Includes bouncy animations and instant visual success messages.
- **Responsive:** Floating navigation menu that works on different screen sizes.

## 🛠️ Tech Stack

- **React:** Functional components and hooks (`useState`, `useEffect`).
- **Vanilla CSS:** Custom styles with CSS Variables and Keyframe animations.
- **SVG:** Used for the interactive Pizza slices.
- **HTML5 Drag and Drop API:** Powering the block-based interactions.

## 🏁 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- npm

### Installation
1. Clone or download the repository.
2. Navigate to the project folder:
   ```sh
   cd fractions
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

### Running the App
Start the development server:
   ```sh
   npm start
   ```
The app will open at `http://localhost:3000`.

## 🍎 Educational Goals
This app is designed to address common hurdles in early fraction learning:
- **Denominator Confusion:** Helping kids realize that $1/8$ is smaller than $1/2$ by showing the physical sizes.
- **Equivalence:** Visually proving that $2/4$ is the same as $1/2$.
- **Number Sense:** Moving from "shapes" to "number line positions" to prepare for school-level math.
